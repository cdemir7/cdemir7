# Selamlar, Ben Cihan 👋
### *AI & Makine Öğrenmesi Mühendisi | Veri Bilimci*

Samsun Ondokuz Mayıs Üniversitesi Bilgisayar Mühendisliği mezunu, Kastamonu Üniversitesi'nde Yapay Zeka alanında yüksek lisans öğrencisiyim.
